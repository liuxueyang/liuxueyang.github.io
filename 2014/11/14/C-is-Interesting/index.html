  • 关于取余。

q = a / b;
r = a % b;
不妨假设b > 0;

我们一般希望满足三点：

1. q * b + r == a
2. 若改变a的正负号，那么这会改变q的符号，但是q的绝对值不变
3. 当b > 0 时，r >= 0 && r < b
