917. Reverse Only Letters

917. Reverse Only Letters

這道題目很簡單啦，給定一個字符串，包含字母和其它字符，返回一個字符串，只把字母倒序，別的字符位置保持不變。基礎的字符串處理題目。我的做法是先把字符提取出來，然後逆序遍歷源字符串，依次替換遇到的字母。 

class Solution {
public:
  string reverseOnlyLetters(string S) {
    string res;
    for (auto x : S) if (isalpha(x)) res += x;
    int j = 0;
    REPV(i, S.length()) {
      if (isalpha(S[i])) { S[i] = res[j++]; }
    }
    return S;
  }
};

int main( void ) {

#ifdef DEBUG
  freopen("917.in", "r", stdin);
#endif

  vector<string> a {
                    "ab-cd",
                    "a-bC-dEf-ghIj",
                    "Test1ng-Leet=code-Q!"
  };
  Solution s;
  for (auto x : a) PRLN(s.reverseOnlyLetters(x));

  return 0;
}

918. Maximum Sum Circular Subarray

918. Maximum Sum Circular Subarray

給定一個循環數組，尋找子數組的最大和。也就是子數組可以從數組的結尾開始、數組的開頭結束。

分兩種情況，一種是子數組不跨越數組結尾，另外一種是子數組跨越數組結尾。

  1. 子數組不跨越數組結尾。就是平常的求子數組最大和的問題。考慮某個數組元素是否要加入到當前數組， dp[i] 表示以第 i 個元素爲結尾的子數組最大和，那麼有 dp[i+1]=max(a[i+1],dp[i]+a[i+1]) ，根據這個等式可以求出以所有元素爲結尾的子數組的最大和，找出最大值就是所求。觀察這個等式，可以發現只有當 dp[i] 爲正的時候才需要累加它，否則 dp[i+1] 的值就是 a[i+1] 本身。因此下一個狀態的值僅僅依賴于上一個狀態的正負，可以節省 dp 數組，只需要一個保存上一個狀態的變量即可。
  2. 子數組跨越數組結尾。也就是子數組最大和是數組的結尾部分和數組的開頭部分，也就是說數組其餘的中間部分是最小和。因此只需要求子數組的最小和，然後用總和減去最小和就是所求。而求數組最小和的思路和求最大和一樣。對於這種情況，所求最小和的子數組不能包含第一個和最後一個元素，因爲跨越數組結尾的最大子數組必須要包含第一個和最後一個元素。

有兩種處理方法：

  • 直接處理。先求 a[0, n-1] 子數組最大和 sum1 ，然後求 a[1, n-2] 子數組最小和 sum2 ，然後 max(sum1, sum-sum2) 即爲所求。 sum 是數組所有元素的總和。
  • 分別求出整個數組的子數組最大和 sum1 和最小和 sum2 。那麼結果就是 max(sum1, sum-sum2) 。考慮特殊情況，如果 sum1 爲負數，說明整個數組都爲負數，此時結果應該是 sum1 而不是 sum-sum2 （也就是 0）。

class Solution {
public:
  int max_sum(VI::iterator s, VI::iterator e) {
    int res = *s, cur = res;
    for (auto i = s + 1; i != e; ++i) {
      cur = max_(cur + *i, *i);
      res = max_(res, cur);
    }
    return res;
  }
  int maxSubarraySumCircular_1(vector<int>& A) {
    int sum1 = max_sum(A.begin(), A.end()), sum = 0;
    for (auto &x : A) { sum += x; x = -x; }
    int sum2 = sum + max_sum(A.begin() + 1, A.end() - 1);
    return max_(sum1, sum2);
  }
  int maxSubarraySumCircular(vector<int>& A) {
    int f = A[0], max_sum = f, min_sum = f, max_cur = 0, min_cur = 0;
    f = 0;
    for (auto x : A) {
      max_cur = max_(max_cur + x, x);
      max_sum = max_(max_sum, max_cur);

      min_cur = min_(min_cur + x, x);
      min_sum = min_(min_sum, min_cur);

      f += x;
    }
    return max_sum > 0 ? max(max_sum, f - min_sum) : max_sum;
  }
};

int main( void ) {

#ifdef DEBUG
  // freopen("918.in", "r", stdin);
#endif

  vector<VI> a {
                {1,-2,3,-2},
                {5,-3,5},
                {3,-1,2,-1},
                {3,-2,2,-3},
                {-2,-3,-1}
  };
  Solution s;
  for (auto &x : a) { PRLN(s.maxSubarraySumCircular(x)); }

  return 0;
}

919. Complete Binary Tree Inserter

919. Complete Binary Tree Inserter

往一個完全二叉樹裏面插入節點，當時沒看懂題目，題目的輸入有點兒迷，不是特別適應這種格式的題目，隨便寫了一下，竟然過了。。 


/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode(int x) : val(x), left(NULL), right(NULL) {}
 * };
 */
struct TreeNode {
  int val;
  TreeNode *left;
  TreeNode *right;
  TreeNode(int x) : val(x), left(NULL), right(NULL) {}
};

class CBTInserter {
public:
  CBTInserter(TreeNode* root) {
    Root = root;
  }

  int insert(int v) {
    TreeNode *c = Root;
    queue<TreeNode*> que; que.push(c);
    while (!que.empty()) {
      c = que.front();
      if (c->left == NULL) {
        c->left = new TreeNode(v);
        return c->val;
      }
      else if (c->right == NULL) {
        c->right = new TreeNode(v);
        return c->val;
      }
      else {
        que.pop();
        que.push(c->left);
        que.push(c->right);
      }
    }
    return -1;
  }

  TreeNode* get_root() {
    return Root;
  }
  TreeNode *Root;
};
