從數組 a 複製 k 個元素到數組 b： 

memcpy(b, a, k * sizeof(T) * k)

其中 T 是數組元素類型。

如果要把數組 a 全部複製到數組 b 中： 

memcpy(b, a, sizeof(a))

memcpy 這個函數在 string.h 中聲明。

memset 函數也在 string.h 中聲明，把數組 a 清零： 

memset(a, 0, sizeof(a))

memset 函數也可以用於多維數組。

介紹實用的字符串處理函數。

3.1 fgetc 和 getchar

fgetc(stdin) 從標準輸入讀取一個字符，返回一個 int ，如果遇到文件結束符，返回 EOF 。相當於 getchar() 函數。使用 getchar() 函數處理字符的時候需要特別小心，尤其注意空格和換行。

陷阱：不同操作系統的回車換行符不一樣。

  • Linux 是 '\n'
  • Windows 是 '\r\n'
  • macOS 是 '\r'

如果在 Linux 中讀取 Windows 格式的文件可能會出現問題：

  • 如果在 Windows 中讀取換行符， getchar 只會讀取到 '\n'，會自動把 '\r' 忽略掉
  • 如果在 Linux 中讀取 Windows 格式的文本文件， getchar 會先讀取到 '\r' ，然後再讀取到 '\n'

所以在不同系統讀取同一個文本文件需要特別注意。

3.2 fgets 和 gets

fgets(buf, MAXN, fin) 讀取完整的一行，讀取字符個數不超過 MAXN - 1，然後自動在末尾添上結束符 '\0'。它遇到換行符就結束讀取，換行符就是 buf 的最後一個字符（再往後就是字符串結束符了）。當這個函數一個字符也沒有讀取到的時候，函數返回 NULL 。

gets(s) 沒有指明讀取的最大字符個數，有緩衝區漏洞。一般不使用。

3.3 ctype 中的幾個實用函數

  • isalpha 判斷字符是否爲大寫字母或者小寫字母
  • isdigit 判斷字符是否爲數字
  • isprint 判斷是否爲可打印字符
  • toupper 轉化爲大寫字母
  • tolower 轉化爲小寫字母

3.4 實現

const int MAXN = 5000 + 20;
char buf[MAXN], s[MAXN], p[MAXN];

int main( void ) {

#ifdef DEBUG
  freopen("3-4_parli_substr.in", "r", stdin);
#endif

  int len = 0;

  while (NULL != fgets(buf, MAXN, stdin)) {
    REP(i, strlen(buf)) {
      if (isalpha(buf[i])) {
        p[len] = i;
        s[len++] = toupper(buf[i]);
      }
    }

    int start = 0, end = 0, res_len = 1;

    REP(i, len) {
      // odd length
      for (int j = 0; i - j >= 0 && i + j < len; j++) {
        if (s[i + j] != s[i - j]) break;
        if (j * 2 + 1 > res_len) {
          res_len = j * 2 + 1;
          start = p[i - j];
          end = p[i + j];
        }
      }

      // even length, take care of index to ensure start <= end
      for (int j = 0; i - j >= 0 && i + j + 1 < len; ++j) {
        if (s[i - j] != s[i + j + 1]) break;
        if (j * 2 + 2 > res_len) {
          res_len = j * 2 + 2;
          start = p[i - j];
          end = p[i + j + 1];
        }
      }
    }

    PRINT1(res_len);
    FOR_1(i, start, end) {
      printf("%c", buf[i]);
    } NL;
  }

  return 0;
}

4.3.1 使用字符串流

int main( void ) {

#ifdef DEBUG
  freopen("getline_test.in", "r", stdin);
#endif

  char s[1000];
  cin.getline(s, 1000, '\n');

  int a, b;

  stringstream ss(s);

  ss >> a >> b;
  PR2(a, b);

  return 0;
}
