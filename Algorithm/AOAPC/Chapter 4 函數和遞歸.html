1.2 應用舉例

1.2.1 組合數

計算組合數，計算過程中使用 int 會溢出。

注意，有的時候雖然計算結果不會溢出，但是計算過程中很可能會溢出。

這題 m, n <= 20，所以使用 long long 就可以了： 

LL factorial(int n)
{
  LL r = 1;
  REP_1(i, n) r *= i;
  return r;
}

int main( void ) {

#ifdef DEBUG
  freopen("4-1-combination.in", "r", stdin);
#endif

  LL m, n;

  while (cin >> m >> n) {
    if (m >= n) PRLN(1);
    else {
      LL res_d = 1;
      REP_1(i, n - m) res_d *= i;
      LL res_u = 1;
      DWN_1(i, n, m + 1) res_u *= i;
      PR2(res_u, res_d);
      PR1(res_u / res_d);
    }
  }

  return 0;
}

1.2.2 孿生素數

判斷是否是素數的時候，使用 x * x < n 有溢出的潛在問題，如果溢出，可能會造成死循環。解決方法是先求平方根，向上取整。

寫函數的時候，使用 assert 來檢查參數，可以更容易發現 BUG。 

bool is_prime(LL x)
{
  if (x == 1) return false;
  assert(x > 1);
  LL m = ceil(sqrt(x));
  FOR_1(i, 2, m) { if (!(x % i)) return false; }
  return true;
}

int main( void ) {

#ifdef DEBUG
  freopen("4-2-prime-pair.in", "r", stdin);
#endif

  LL m;

  while (cin >> m) {
    DWN_1(i, m - 2, 3) {
      if (is_prime(i) && is_prime(i + 2)) {
        PR2(i, i + 2);
        break;
      }
    }
  }

  return 0;
}
