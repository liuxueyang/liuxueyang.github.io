##Some common lisp simple syntax Here is some common lisp syntaxes.

define local variables:

(let ((a 5) (b 6))
	(+ a b))

define local functions:

(flet ((f (n) (+ n 1))
	   (g (n) (+ n 2)))
	  (g (f 3)))

To make function names available in defined functions, we can use the labels command.

(labels ((a (n) (+ n 5))
		 (b (n) (+ (a n) 6)))
	(b 10))
