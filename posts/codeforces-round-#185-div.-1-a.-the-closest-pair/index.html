题目链接：http://codeforces.com/problemset/problem/311/A 题目大意： 　　题目很短，不解释了。^_^ 题目思路： 　　开始感觉很难，没思路。看了解题报告，发现原来是道水题嘛。程序里面有个break语句，要让循环次数达到最大，并且循环次数容易计算，只需要让这个break;永远不会执行就可以了。也就是说，可以让p[j].x - p[i].x >= d 永远不成立，只需要让p[j].x - p[i].x总等于0就可以了！因为题目让生成任意一组符合条件的数据嘛，所以，可以让产生的所有的点的横坐标都是一样的就可以了。


#include <cstdio>
#include <cstdlib>
#include <cstring>
#include <iostream>
#include <cmath>
using namespace std;
#define LL long long 
int main(void) {
#ifndef ONLINE_JUDGE
  //freopen("185_c.in1", "r", stdin);
#endif
  LL n, k; scanf("%I64d%I64d", &n, &k);
  if (n*(n-1)/2 <= k){
    printf("no solution\n");
  }
  else {
    for (int i = 0; i < n; ++i) {
       printf("1 %d\n", i);
    }
  }
  return 0;
}
