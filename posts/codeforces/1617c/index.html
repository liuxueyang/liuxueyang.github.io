CodeForces 1617C Paprika and Permutation

December 13, 2023 阅读时间：2 分钟
CodeForces
Math
  • Paprika and Permutation
  • tutorial

对于恰好就在 $[1, n]$ 范围内的数字，可以不用操作。因为一次的模数 $x$ 是任意取的，考虑 $a_i \bmod x$ 的取值范围，当 $a_i < x$ 的时候，$a_i \bmod x = a_i$；当 $a_i > x$ 时，$a_i \bmod x \le \lfloor \frac{a_i - 1}{2} \rfloor$，同时也能发现此时 $a_i \bmod x$ 的值域恰好是 $[1, \lfloor \frac{a_i - 1}{2} \rfloor]$ 中的所有整数。所以把剩余的 $a_i$ 从小到大排序之后，计算出值域，检查它们能不能生成 $[1, n]$ 中剩余的值即可。

const int N = 100010;
int t, n, a[N], b[N], c[N];

int main(void) {

  std::ios::sync_with_stdio(false);

  cin >> t;

  while (t--) {
    cin >> n;

    For1(i, 1, n) { cin >> a[i]; }

    int len = 0;
    memset(vis, false, sizeof vis);

    For1(i, 1, n) {
      if (a[i] >= 1 && a[i] <= n && !vis[a[i]]) {
        vis[a[i]] = true;
      } else {
        b[++len] = a[i];
      }
    }

    int len1 = 0;
    For1(i, 1, n) {
      if (!vis[i]) {
        c[++len1] = i;
      }
    }

    sort(b + 1, b + 1 + len);
    bool flag = true;

    For1(i, 1, len) {
      int top = (b[i] - 1) / 2;
      if (top < c[i]) {
        flag = false;
        break;
      }
    }

    if (flag)
      cout << len << '\n';
    else
      cout << "-1\n";
  }

  return 0;
