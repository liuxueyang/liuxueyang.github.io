1.函数 举个例子：判断一个数字是奇数还是偶数


-module(rev).
-export([even/1]).

even(Value) ->
    if 
        Value rem 2 == 1 -> odd;
        Value rem 2 == 0 -> even
end.

文件名和模块名要相同。在终端中引用函数的时候，先编译，c(rev). 然后引用函数 rev:even(N).  其中N是一个已经赋值的变量。 上面的函数的另一种写法：  


-module(rev).
-export([even/1]).

even(Value) ->
    case Value rem 2 of
        1 -> odd;
        0 -> even
    end.
