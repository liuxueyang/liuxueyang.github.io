nefu118 n!后面有多少个0 数论

Mar 30, 2013 · 1 minute read · Comments

数论题。统计n!里面有多少个素数5即可。公式：[n/p] + [n/p^2] + [n/p^3] + …… 然后发现一个很无语的问题，nefu上面不能用这个：


#ifndef ONLINE_JUDGE
  freopen("nefu118.in", "r", stdin);
#endif

要不要这样啊，别的OJ可都是可以用的好不……


#include <iostream>
#include <cstdio>
#include <cstdlib>
#include <cmath>
using namespace std;
int main(void){
  int n, m;
  scanf("%d", &m);
  while (m--){
    scanf("%d", &n);
    int t = 5, cnt = 0;
    while (t <= n){
       cnt += n/t; t *= 5;
    }
    printf("%d\n", cnt);
  }
  return 0;
}
