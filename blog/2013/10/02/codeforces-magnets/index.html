codeforces Magnets

Oct 2, 2013 · 1 minute read · Comments

link: http://codeforces.com/contest/344/problem/A

这道题目很简单。

把输入的01 和10 当做整数，如果相邻两个数字相等的话，那么就属于同一组，否则，就新增加了一组。

#include <cstdio>
#include <cstdlib>
#include <cstring>
using namespace std;
int main(void)
{
  #ifndef ONLINE_JUDGE
  freopen("in.txt", "r", stdin);
  #endif // ONLINE_JUDGE
  int n;
  while (~scanf("%d", &n))
  {
    int tmp;
    int pre = -1;
    int cnt = 0;
    while (n--)
    {
      scanf("%02d", &tmp);
      if (tmp != pre) cnt++;
      pre = tmp;
    }
    printf("%d\n", cnt);
  }
  return 0;
}
