- [Unstable String](https://vjudge.net/problem/CodeForces-1535c)

有意思的 $DP$ 题目。如果 $s_i$ 是 $?$，它是 $1$ 还是 $0$ 取决于 $s_{i - 1}$，因此想到 $dp$，设 $d_{0, i}$ 表示以 $s_i$ 为结尾，并且 $s_i = 0$ 的合法的最长连续子字符串的个数，$d_{1, i}$ 同理。最终答案就是 $\sum max(d_{0, i}, d_{1, i})$

const int N = 200100;

string s;
int n, d[2][N];
void solve() {
  cin >> s;
  memset(d, 0, sizeof d);

  n = SZ(s);
  if (s[0] == '?')
    d[0][0] = d[1][0] = 1;
  else if (s[0] == '1')
    d[1][0] = 1;
  else
    d[0][0] = 1;

  For(i, 1, n) {
    if (s[i] == '?') {
      d[0][i] = 1 + d[1][i - 1];
      d[1][i] = 1 + d[0][i - 1];
    } else if (s[i] == '1') {
      d[0][i] = 0;
      d[1][i] = d[0][i - 1] + 1;
    } else {
      d[1][i] = 0;
      d[0][i] = d[1][i - 1] + 1;
    }
  }

  ll ans = 0;
  For(i, 0, n) { ans = ans + max(d[0][i], d[1][i]); }

  cout << ans << '\n';
}
