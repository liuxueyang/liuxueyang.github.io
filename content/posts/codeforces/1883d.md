- [In Love](https://vjudge.net/problem/CodeForces-1883d)

要判断是否存在两个区间不相交，只需要找到最小的 $r_i$ 和最大的 $l_j$，如果满足 $r_i < l_j$，那么就存在。

int q, l, r;

bool check() {
  bool mark = false;
  if (nonempty(r_s)) {
