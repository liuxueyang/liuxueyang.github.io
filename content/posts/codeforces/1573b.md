- [Swaps](https://vjudge.net/problem/CodeForces-1573b)

考虑 $a_1$ 不动，只需要在 $b$ 中找到第一个 $b_i > a_1$ 的即可。这个想法推广一下就可以得到，对于每个 $a_i$，在 $b$ 中找到第一个 $b_j > a_i$。剩下的问题就是如何快速在 $b$ 找到第一个 $> a_i$ 的值，根据 $a, b$ 数组中值的特殊性，这可以使用后缀和思想优化。

const int N = 100100;
int n, a[N], b[N];

void solve() {
  cin >> n;

  For1(i, 1, n) {

  For1(i, 1, n) {

    b1[i] = PII(b[i], i);

  sort(a1 + 1, a1 + 1 + n);
  sort(b1 + 1, b1 + 1 + n);

  map<int, int> m;
  m[a1[n].f1] = a1[n].f2;
  m[b1[n].f1] = b1[n].f2;

    ckmin(a1[i].f2, a1[i + 1].f2);
