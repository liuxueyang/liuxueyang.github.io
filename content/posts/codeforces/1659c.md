- [Line Empire](https://vjudge.net/problem/CodeForces-1659c)

考虑基地的最终位置，如果最终的位置在 $x_i$，那么在把基地拉到 $x_i$ 之后，就再也不拉基地了，此时之后的代价可以计算出来。再考虑之前的拉基地的情况，可以发现攻打一个城市就拉一次基地的代价，总是比先攻打几个城市再拉基地的代价要小。因此在 $x_i$ 之前，每次攻打一个基地就拉一次基地，在 $x_i$ 之前的总代价也可以方便地计算出来。

枚举基地最终的位置 $x_i$，使用 $O(1)$ 计算出总代价。

const int N = 200100;
int n;

void solve() {
  cin >> n >> a >> b;

  x[0] = 0;
  For1(i, 1, n) { cin >> x[i]; }
  p[n + 1] = 0;
  Rof1(i, 1, n) { p[i] = x[i] + p[i + 1]; }

  ll ans = INFL;

  For1(i, 0, n) {
    ll left = (x[i] - x[0]) * (a + b);
    ll right = b * (p[i + 1] - x[i] * (n - i));
    ckmin(ans, left + right);
  }

  cout << ans << '\n';
}
