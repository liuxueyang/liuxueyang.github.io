- [Hamiltonian Wall](https://vjudge.net/problem/CodeForces-1766c)

可以发现从一个 B 开始，只能向右、上、下走，并且应该优先走上、下的情况，这样才不会出现重复走到 B 的情况。由于起点不确定，但是只有两个。两次 DFS 就可以了，搜索完之后检查是不是所有的 B 都访问到了。

void dfs(int x, int y) {
  vis[x][y] = true;
  int x1, y1;

  For(i, 0, 3) {
    x1 = x + dir[i][0];
    y1 = y + dir[i][1];

    if (check(x1, y1)) {
      dfs(x1, y1);
      return;
    }
  }
}
