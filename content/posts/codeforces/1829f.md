统计每个点的度数，难点在于边界条件。

|degree|count|
|---|---|
|$x$ | 1|
|1|$xy$|
|$y+1$| $x$|

边界条件是 $x = y + 1$

const int N = 210, M = 50010;

        if (de != 1) {
          resy = de - 1;
          resx = cnt - 1;
          break;
        }
      }
    }

    cout << resx << ' ' << resy << '\n';
  }

  return 0;
}
