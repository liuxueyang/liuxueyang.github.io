- [Milena and Admirer](https://vjudge.net/problem/CodeForces-1898b)
- [tutorial](https://www.luogu.com.cn/problem/solution/CF1898B)

考虑两个数字的情况，可以发现尽量把 $a_i$ 分解成 $\frac{a_i}{2}$ 是最优的，推广可以猜到，如果要把 $a_i$ 分解成 $k$ 个数字，其中 $k > 2$，那么每个数字的大小是 $\frac{a_i}{k}$，也就是分解得到的数字尽量接近。注意这里不能取余，因为取余不是最优的，它会让第一个数字过小。

    Rof1(i, 1, n - 1) {
      if (cur == 1) {
        res += a[i] - 1;
      } else if (a[i] > cur) {
        int y = (a[i] + 1) / 2;
        if (y <= cur) {
          cur = a[i] / 2;
          res++;

          y = cur;

            res += cnt;
            cur = a[i] / (cnt + 1);
          } else {
            res += a[i] / y - 1;
            cur = y;
          }
        }
      } else {
        cur = a[i];
      }
    }

    cout << res << '\n';
  }
